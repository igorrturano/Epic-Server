# Methods de Armaduras

## Gets
* .getSlot(item)
	
		Pega o Slot do Item no .cfg.
		
		Retorna: "Shield" ou Coverage do item.


* .getType(item)
	
		Pega o Tipo de Proteção da armadura.
		
		Retorna: TipoProtecao que está no .cfg.


## Checks

* .isShield(item)

		Verifica se o item é um Escudo.
		
		Retorna: 1 ou 0.
